# 图标文件说明

此目录需要包含以下三个PNG格式的图标文件：

- `icon16.png` - 16x16 像素，用于扩展工具栏
- `icon48.png` - 48x48 像素，用于扩展管理页面
- `icon128.png` - 128x128 像素，用于Chrome Web Store

## 创建图标的方法

### 方法1：使用在线工具
1. 访问 https://www.favicon-generator.org/ 或类似工具
2. 上传你的图标图片
3. 下载生成的PNG文件
4. 重命名并放置到此目录

### 方法2：使用设计工具
使用 Photoshop、GIMP、Figma 等工具创建：
- 设计一个火箭或跳转相关的图标
- 导出为PNG格式
- 确保尺寸正确（16x16, 48x48, 128x128）

### 方法3：临时使用占位符
在开发阶段，你可以：
1. 创建一个简单的纯色PNG图片
2. 或者从网上下载免费的图标
3. 推荐使用与"跳转"、"导航"相关的图标

## 图标设计建议

- 使用简洁的设计
- 使用与"跳转"、"导航"、"链接"相关的元素（如箭头、火箭等）
- 确保在小尺寸下仍然清晰可见
- 使用与扩展主题色（紫色渐变）相配的颜色
