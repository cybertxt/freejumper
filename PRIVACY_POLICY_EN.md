# FreeJumper Privacy Policy

**Last Updated:** 2025

## Overview

FreeJumper ("this extension") is a Chrome browser extension that allows users to select text on web pages and jump to custom URLs via the right-click context menu. This privacy policy explains how we collect, use, and protect your information.

## Data Collection

**This extension does not collect, store, or transmit any user data.**

### Local Storage

This extension only stores the following information locally in your browser:
- **URL Template Configuration**: Your configured URL templates and aliases
- **User Settings**: Extension configuration options

This data is stored only in your browser using Chrome's `chrome.storage.sync` API, which synchronizes to other devices where you are logged into your Chrome account.

### Information Not Collected

This extension **does not** collect any of the following information:
- Personal information (name, email, phone number, etc.)
- Browsing history
- Web page content
- IP addresses
- Device information
- Location information
- Any other personal data

## Data Transmission

**This extension does not send data to any servers.**

All functionality is performed locally in your browser and does not communicate with any external servers.

## Third-Party Services

This extension does not integrate any third-party analytics services, advertising services, or other data collection services.

## Permission Explanations

This extension requests the following permissions, used solely to provide core functionality:

- **`storage`**: Store your URL template configurations
- **`activeTab`**: Access the current tab to retrieve selected text
- **`tabs`**: Open URLs in new tabs
- **`contextMenus`**: Create right-click menu options
- **`<all_urls>`**: Enable the extension to work on all web pages

These permissions are used only to provide the extension's core functionality and are not used to collect or transmit data.

## Data Security

Since this extension does not collect or transmit any data, there are no data breaches or security risks. Your configuration data is stored only in your browser and is protected by the Chrome browser.

## Children's Privacy

This extension does not collect any information, so it does not involve children's privacy issues.
