# FreeJumper Chrome 扩展

一个强大的Chrome扩展，允许你选中网页上的文字后，将其作为自定义URL的一部分，直接在新标签页中打开。

## 功能特点

- 🎯 **多URL模板配置** - 支持配置多个URL模板，每个模板可设置别名
- 🖱️ **右键菜单选择** - 选中文字后，右键菜单中显示所有配置的跳转选项
- 🚀 **快速跳转** - 从右键菜单选择目标URL，直接在新标签页打开
- 🎨 **美观的UI** - 现代化的界面设计，使用体验流畅
- 📝 **实时预览** - 在配置页面可以实时预览所有模板生成的URL
- 🔧 **预设模板** - 内置常用网站的URL模板，一键添加
- ✏️ **灵活管理** - 支持添加、编辑、删除模板，自由定制

## 安装方法

### 第一步：生成图标（首次安装需要）

1. 在浏览器中打开 `icons/generate-icons.html` 文件
2. 点击每个尺寸旁边的"下载"按钮，下载三个图标文件
3. 将下载的文件重命名为：
   - `icon16.png`
   - `icon48.png`
   - `icon128.png`
4. 将这三个文件放置到 `icons/` 目录中

> **注意**: 如果没有图标文件，扩展仍然可以工作，但工具栏可能不会显示图标。

### 第二步：安装扩展

1. 打开Chrome浏览器，进入扩展管理页面：
   - 在地址栏输入 `chrome://extensions/`
   - 或者点击右上角三点菜单 → 更多工具 → 扩展程序

2. 启用"开发者模式"（右上角开关）

3. 点击"加载已解压的扩展程序"

4. 选择本项目的文件夹

5. 安装完成！

## 使用方法

### 基本使用

1. 在任意网页上选中一段文字
2. 右键点击选中的文字
3. 在右键菜单中找到"🚀 FreeJumper 跳转到"选项
4. 选择你想要跳转的目标（如"Google搜索"、"百度搜索"等）
5. 选中的文字会被插入到对应的URL模板中，并在新标签页打开

### 配置URL模板

1. 点击浏览器工具栏中的FreeJumper图标，打开弹出窗口
2. 点击"打开配置页面"按钮
3. 在配置页面中：
   - 点击"+ 添加模板"按钮添加新的URL模板
   - 为每个模板设置别名（显示在右键菜单中）
   - 输入URL模板（必须包含 `{text}` 占位符）
   - 可以使用预设的示例模板快速添加（Google搜索、百度搜索、GitHub搜索等）
   - 实时预览所有模板生成的URL
   - 点击"保存配置"保存设置
   - 可以删除不需要的模板（至少保留一个）

### URL模板示例

配置页面提供了快速添加按钮，你也可以手动添加：

- **Google搜索**: `https://www.google.com/search?q={text}`
- **百度搜索**: `https://www.baidu.com/s?wd={text}`
- **GitHub搜索**: `https://github.com/search?q={text}`
- **Bing搜索**: `https://www.bing.com/search?q={text}`
- **Google翻译**: `https://translate.google.com/?sl=auto&tl=zh-CN&text={text}`
- **Wikipedia**: `https://www.wikipedia.org/wiki/{text}`
- **自定义搜索**: `https://example.com/search?query={text}`

### 多模板管理

- **添加模板**: 点击"+ 添加模板"按钮，填写别名和URL模板
- **编辑模板**: 直接在输入框中修改别名或URL模板
- **删除模板**: 点击模板右上角的删除按钮（至少保留一个模板）
- **快速添加**: 点击预设模板按钮，一键添加常用网站模板

## 开发说明

### 技术栈

- Chrome Extension Manifest V3
- 原生JavaScript（无框架依赖）
- Chrome Storage API（同步存储配置）

### 主要功能实现

1. **右键菜单**: 使用 `chrome.contextMenus` API 创建右键菜单选项
2. **多模板管理**: 支持配置多个URL模板，每个模板有独立的别名和URL
3. **URL生成**: 将选中的文字替换URL模板中的 `{text}` 占位符
4. **新标签页打开**: 使用 `chrome.tabs.create` API 在新标签页打开URL
5. **配置管理**: 使用 `chrome.storage.sync` 同步存储用户配置，支持多设备同步
6. **动态菜单更新**: 配置更改后自动更新右键菜单

## 注意事项

- URL模板必须包含 `{text}` 占位符，否则无法正常工作
- 选中的文字会自动进行URL编码，确保特殊字符正确处理
- 配置会同步到你的Chrome账户，在不同设备间自动同步
- 至少需要配置一个有效的URL模板
- 模板别名可以为空，如果为空将显示为"模板 N"（N为序号）
- 保存配置后，右键菜单会在几秒内自动更新
